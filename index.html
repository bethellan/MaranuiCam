About This Data

The hourly forecast table and Surfability score use live data for Lyall Bay, Wellington.

  • Open-Meteo Forecast API – wind speed, gusts, direction, and rainfall.
  • Open-Meteo Marine API – wave height, period, and direction.
  • Open-Meteo Tide API – tide height estimates.
  • Sunrise-Sunset.org – sunrise and sunset times.

Data updates every 30 minutes and covers the next 24 hours in local time. If live data is unavailable, a built-in offline dataset ensures the table still displays.

Surfability Calculation

Wave Height Scoring:
< 0.5m = 3 (too small)
0.5–1.5m = 5–7.5 (good for beginners)
1.5–2.5m = 7–9.5 (great for experienced)
> 2.5m = 9 (excellent big waves)

Cleanliness Factors:
+0.5 for wave period > 10s (clean)
+0.2 for wave period 7–10s (decent)
−0.3 for wave period < 5s (choppy)
+0.3 for offshore winds (aligned)
−0.3 for onshore winds (messy)

Adjustments:
−((wind − 20)/5) if wind > 20 km/h
−2 if rain > 0.5 mm/hr
+1 if wind dir 200–340° (offshore)

8–10: Excellent · 5–7.9: Fair · <5: Poor

Note: This formula rewards bigger waves that experienced surfers and kids prefer!
