Maranui SLSC – Lyall Bay Webcam (v2, branded)
=================================================

What this is:
- An installable web app (PWA) themed for Maranui SLSC that embeds the YouTube Live webcam.
- iOS-friendly: Add to Home Screen, tap ▶ Play, then ⛶ for fullscreen.

Logos used:
- assets/logos/maranui-script.png (header)
- assets/logos/maranui-cap.png (hero background motif)
- assets/logos/maranui-wings.png (header watermark + app icon)

How to use:
1) Upload all files to a single folder on your web host (GitHub Pages works well).
2) Visit the page on your iPhone/iPad (Safari), tap Share → Add to Home Screen.
3) Open from the Home Screen. Tap ▶ Play to start (iOS requires a user gesture).

Change stream:
- Edit VIDEO_ID in script.js or replace the iframe src in index.html.

Brand colours (style.css :root):
- --lifeguard-red: #C8102E
- --rescue-yellow: #FFD200
- --club-blue: #005CAB
